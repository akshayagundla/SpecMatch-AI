# SpecMatch AI 🚀
AI-powered electronics recommendation platform that uses Gemini AI, Retrieval-Augmented Generation (RAG), and semantic search to match users with the best products based on their requirements.

## Features

- AI-powered product recommendations
- Gemini AI integration
- RAG-based knowledge retrieval
- Semantic product matching
- Conversational chat interface
- Product comparison support
- Session history tracking
- Supabase backend integration

## Architecture

User Query
    ↓
Gemini AI
    ↓
RAG Search Engine
    ↓
Product Matching Layer
    ↓
Recommendation Results

## Tech Stack

Frontend:
- React
- TypeScript
- TanStack Router
- Vite

Backend:
- Supabase
- Node.js

AI:
- Gemini AI
- Vector Search
- RAG Pipeline
